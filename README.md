## 🚀 Quick Start

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open browser
http://localhost:3000
```

## 🌟 Features

- ✅ **9 Pages** - Home, About, Contact, Gallery, Rooms, Restaurant, Safari Tours
- ✅ **Responsive Design** - Mobile, tablet, desktop optimized
- ✅ **SEO Optimized** - Meta tags, Open Graph, structured data
- ✅ **Booking System** - EmailJS integration for reservations
- ✅ **Image Gallery** - Filterable masonry gallery
- ✅ **Animations** - AOS animations, smooth scrolling
- ✅ **Forms** - Contact & booking forms with validation

---

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (Pages Router)
- **UI Library:** React 18, React Bootstrap
- **Styling:** Bootstrap 5, Custom CSS, Emotion
- **Forms:** Formik, Yup validation
- **Email:** EmailJS
- **Icons:** React Icons, Bootstrap Icons
- **Animations:** AOS, Animate.css
- **Carousel:** Swiper
- **Gallery:** React Masonry CSS
